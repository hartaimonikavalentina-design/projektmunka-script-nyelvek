## 📌 Leírás

Ez a projektmunka meteorológiai adatok feldolgozását és vizualizációját mutatja be Python és Jupyter Notebook segítségével.  
A cél az volt, hogy a nyers adatokból interaktív grafikonok és áttekinthető elemzések készüljenek.

A notebook tartalmazza:

- az adatok beolvasását és tisztítását  
- statisztikai vizsgálatokat  
- interaktív grafikonokat  
- a teljes munkafolyamatot dokumentáló kiegészítő fájlokat (PNG, HTML stb.)

A projekt a **PPKE Script nyelvek** kurzus keretében készült.

## 📁 Fájlok a repóban

- `projektmunka.ipynb` – a teljes, futtatható notebook  
- `adatok_meteorologia.csv` – a felhasznált adatok  
- `app.py` – a Streamlit alkalmazás  
- `requirements.txt` – a szükséges csomagok listája  
- `adatok_meteorologia.zip` – tömörített adatfájlok
